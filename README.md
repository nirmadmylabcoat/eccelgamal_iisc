Implements a commitment scheme:

- Uses a fixed generator `G` and a derived second base `H'`:
  \[
  C = mG + H'
  \]
- H' is generated by hashing a random `r` and mapping to a curve point

- **Hiding**: Random `H'` masks `m`, so no info is leaked
